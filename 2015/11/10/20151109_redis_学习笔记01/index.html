安装redis

我知道，类似这样的教程，网上会有一大堆，有比我的好的，我没有别的目的，只是希望自己能够对之前做过的事情，有个记录。即便忘记了，我也会来看看自己写的这些东西来看看自己哪方面理解的不对。纸上得来终觉浅，绝知此事要躬行。

解压、make

进入redis的根目录使用如下命令

make

centos6.5镜像安装gcc

尝试make

make

会发现接着报错，这是因为之前make的时候报的错，还存在， 这时候就需要：

make distclean

然后执行：

make

发现类似黑客的一系列代码之后就出现了:

Hint: It's a good idea to run 'make test' ;)
这时候我们的redis就已经安装成功了。

至于这个make test 嘿嘿，我们之后还会研究。

make install

安装成功后， 会在/usr/local/bin下面出现一堆文件：redis-server,redis-cli等
